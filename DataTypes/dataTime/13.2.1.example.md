## ‚úÖ **Table Creation:**
```sql
CREATE TABLE IF NOT EXISTS datetype_example (
    currents_date DATE,                                 -- Stores date in YYYY-MM-DD format
    currents_time TIME,                                 -- Stores time in HH:MM:SS format
    currents_timeStamp TIMESTAMP,                       -- Stores timestamp (date + time)
    currents_year YEAR,                                 -- Stores 4-digit year (YYYY)
    creted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,      -- Automatically sets current timestamp on insert
    update_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP -- Updates timestamp on modification
);
```

## ‚úÖ **Explanation:**
1. **currents_date** ‚Äì Stores the date in `YYYY-MM-DD` format.
2. **currents_time** ‚Äì Stores the time in `HH:MM:SS` format.
3. **currents_timeStamp** ‚Äì Stores the date and time in `YYYY-MM-DD HH:MM:SS` format.
4. **currents_year** ‚Äì Stores a 4-digit year between `1901` and `2155`.
5. **creted_at** ‚Äì Automatically stores the current timestamp when a row is inserted.
6. **update_at** ‚Äì Automatically updates to the current timestamp when a row is modified.
